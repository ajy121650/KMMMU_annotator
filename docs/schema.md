# Annotation Schema (Draft)

## Key

- `item_id` (원본 문제 식별자)

## Human Label Fields

- `human_judgement`: `correct | incorrect | no_answer`
- `question_type_match`: `match | mismatch | unsure`
- `annotator_id`: string
- `time_spent_sec`: number
- `comment`: string (optional)
- `updated_at`: ISO datetime
